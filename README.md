## 📋 What is email-verification-api?

The email-verification-api is a simple tool that helps you check if an email address is real and usable. It can quickly tell you if an email is valid, if it comes from a free service like Gmail or Yahoo, or if it's disposable and temporary. It also gives you details about the owner, such as their name and gender.

This tool works as a REST API, which you don’t need to worry about if you just want to use the program to verify emails. It is designed to help you confirm email addresses in bulk, avoid sending emails to invalid or fake addresses, and make your contacts list cleaner and more reliable.
